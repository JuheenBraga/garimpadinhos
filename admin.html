🚀 Publicar site

Escolha como quer publicar as alterações:

⬇ Download manual Simples

Baixa o index.html atualizado e você faz upload no Cloudflare Pages. Leva 1 minuto.

⚡ Publicação automática via GitHub Recomendado

Configure uma vez e o site atualiza sozinho sempre que publicar. Sem precisar baixar nada.

Como configurar (1 vez só):
1) Crie conta em github.com
2) Crie um repositório público chamado garimpadinhos
3) Conecte esse repositório ao Cloudflare Pages em Workers & Pages → Create → Pages → Connect to Git
4) Gere um token em GitHub → Settings → Developer settings → Tokens (classic) com permissão repo
5) Cole os dados abaixo e clique em Testar.
✓ salvo
  1. Clique em "Baixar index.html"
  2. Acesse cloudflare.com → Workers & Pages → seu projeto
  3. Vá em Deployments → Upload assets e arraste o novo index.html
  4. Pronto — site atualizado! 🎉
